# Java Practice for Software Quality Assurance (SQA)

## 📌 Overview
This repository contains my Java practice codes focused on building strong programming fundamentals for Software Quality Assurance (SQA). 
The main goal is to improve problem-solving ability, understand core programming logic, and build a strong foundation for test automation.

## 🎯 Objectives
- Strengthen Java fundamentals  
- Improve logical thinking and problem-solving skills  
- Understand programming behavior for testing purposes  
- Build foundation for automation testing tools like Selenium  
- Prepare for real-world SQA roles  

## 🛠️ Topics Covered
- Java Basics (Syntax, Variables, Data Types)  
- Conditional Statements (if-else, switch)  
- Loops (for, while, do-while)  
- Functions / Methods  
- Arrays  
- Strings  
- Object-Oriented Programming (OOP)  
- Exception Handling (coming soon)  
- File Handling (coming soon)  

## 🧪 SQA Relevance
This repository is not just for coding practice. It is designed to support Software Quality Assurance learning by:

- Understanding application logic  
- Writing test scenarios using code thinking  
- Improving debugging skills  
- Preparing for automation testing (Selenium, TestNG, etc.)  
- Building strong analytical thinking for QA roles
